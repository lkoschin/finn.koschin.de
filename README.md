# Finn.koschin.de

Willkommen beim Webseiten-Projekt von **Finn**!

Dieses Repository enthaelt die Quelltexte unter  
**https://finn.koschin.de**

## Entwicklung

1. Repository klonen:
   ```
   git clone https://github.com/lkoschin/finn.koschin.de.git
   ```

2. Aenderungen pushen (Auto-Deploy alle 5 Min, Mo-Fr 8-20 Uhr):
   ```
   git add .
   git commit -m "Meine Aenderung"
   git push
   ```

## Technik

- **Framework:** Symfony 7.2 (PHP 8.3)
- **Datenbank:** MySQL 8.0
- **Webserver:** Apache 2.4 auf Ubuntu 24.04
- **SSL:** Lets Encrypt (Auto-Erneuerung)
